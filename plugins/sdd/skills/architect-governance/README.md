# architect-governance

The SDD default for the **Architect** actor governance — the structural-fit bar (no duplication, no conflict, contained complexity). Loaded by the plan-producer and impl-producer (to self-align) and by the gate's Architect-backward face (to judge structure).

Resolved like a role: a plugin may bind its own Architect governance via the registry `governances.architect` value; when that is `null`, this default loads. Reference content only — no rationale prose.
