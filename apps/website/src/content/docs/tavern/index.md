title: Tavern
description: Browse and install crews — persona-gateway catalog entries tagged "crew".
---

The **Tavern** is the storefront for **crews** — catalog entries that ship an installable persona
gateway skill, marked by a reserved `crew` tag in the entry's `tags[]`. It lists and points to
install only; it does not deploy, tune, or recruit a crew (those are cyberfleet concerns).

Browse the same roster from the CLI:

```bash
npx cyberplace@$(npm view cyberplace version) tavern
```

Filter by free text:

```bash
npx cyberplace@$(npm view cyberplace version) tavern <query>
```
