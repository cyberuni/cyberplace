**Trigger:** invoked when **reconfiguring an automaton already in the fleet** — model/effort, governance or loadout re-chip, leash, or a whole-unit hot-swap. Not for recruiting a new crew ([Crimp](/cyberfleet/crimp/)), deploying a ship instance ([Operator](/cyberfleet/operator/)), or authoring a brand-new skill from scratch.

Once a crew is aboard, the **Tuner** adjusts how it runs — the guidance it follows, the model it uses, how hard it thinks, and how much leash it has before it checks back with you.
