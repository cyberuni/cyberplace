**Trigger:** invoked when **building or reconfiguring an automaton** — a human-facing gateway persona skill. Author a brand-new one from scratch, or change an existing one's model/effort, governance or loadout re-chip, leash, or a whole-unit hot-swap. Not for recruiting an already-published crew ([Crimp](/cyberfleet/crimp/)), deploying a ship instance ([Operator](/cyberfleet/operator/)), or authoring a plain workflow skill that is not an automaton.

The **Mechanic** works the bench: it stamps a fresh automaton when the fleet needs one, and once a crew is aboard it adjusts how that crew runs — the guidance it follows, the model it uses, how hard it thinks, and how much leash it has before it checks back with you.
