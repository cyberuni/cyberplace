# design/ — the cyberspace foundation model

The rules/model: what makes agent config **harness-agnostic** (the write-once → cross-vendor mapping), the
baseline-config conventions a project starts from, and the boundary with ACED (cyberspace bootstraps and
distributes; ACED authors and evaluates). Decisions and their rationale live in
[`decisions/`](./decisions/README.md). Behaviors live in the capability folders.
